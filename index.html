About Me

Systems Engineer specialized in Game Programming and UI Development. Graduated in 2024 as a Systems Engineer, I began my journey in software development with a Front-End internship in 2023, using Flutter and Dart to build a web app. In October 2024, I joined U-Tad University to specialize in Game Programming. Since then, I have worked for 8 months at Zulo Interactive, a student-led studio, contributing to Gameplay and UI systems for Lady Umbrella, a title set to release on Steam in October 2025. My focus is on creating intuitive and visually cohesive gameplay experiences through solid technical and UI design foundations.
